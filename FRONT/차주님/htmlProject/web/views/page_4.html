링크 관련 태그 : a (anchor 의 줄임말)

속성(attribute)

  1. href="연결할페이지파일명/웹사이트url" : 필수 속성임.
  2. name="위치이름" : 페이지에 북마크 표시할 때 사용
  3. target="프레임이름/윈도우이름" : 연결한 페이지 출력창
  4. title="링크에 대한 설명"
  • 글자로 다른 페이지 연결
  • 그림으로 다른 페이지 연결
  • 웹사이트 연결 오라클
  • 북마크 표시한 위치로 화면이동
