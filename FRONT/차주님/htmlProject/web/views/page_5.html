표(table) 태그 연습 페이지

table 태그는 내용의 규칙적인 배치를 위해 주로 사용됨.

테이블 속성
  • width 속성 : 테이블의 가로 너비 지정
  • height : 테이블 세로 높이
  • border : 테두리선
  • align : 테이블 위치
  • cellspacing : 칸과 칸(셀:cell) 사이의 간격
  • cellpadding : 칸 안 여백
