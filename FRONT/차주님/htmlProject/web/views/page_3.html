목록 관련 태그 연습

목록 태그는 ol(order list : 순번있는 목록)과 ul(unorder list : 순번없는 목록)과 dl(define list : 정의목록)이 있음.

순번이 매겨지는 목록 : OL 태그

  1. 자바
  2. 오라클
  3. JDBC
  4. HTML5
  5. CSS3

순번이 없는 목록 : UL 태그, 뷸릿기호로 표시됨

  • 자바스크립트
  • 제이쿼리
  • 서블릿
  • JSP
  • Ajax

용어와 용어 설명을 목록으로 표현하는 태그 : dl

HTML5
W3C (세계 웹표준협회)에서 웹서비스에 사용되는 문서 제작에 사용하는 표준언어로 지정한 마크업(Markup) 언어이다.
CSS3
HTML5와 함께 새로운 웹표준으로 지정된 스타일시트 언어이며, 태그로 표현된 웹 문서에 디자인을 입히는 용도로 사용된다.
JavaScript
전세계 모든 웹브라우저에 내장된 스크립트 언어로써, html 문서 안에서 작성되며 정적인 태그에 동적인 효과를 주는 목적으로 사용된다.
